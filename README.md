安装vue-cli

先安装node和npm 
1.全局安装vue-cli

```
npm install --global vue-cli 
```

2.由于国内npm较慢,选择安装ali的,以后就是用cnpm

```
npm install -g cnpm --registry=https://registry.npm.taobao.org 
```

3.新增一个空项目后,使用vue-cli进行初始化

```
vue init webpack demo01(项目名) 
```
